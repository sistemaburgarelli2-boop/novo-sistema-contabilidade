# Components

Componentes visuais reutilizaveis.

Regras:

- Componentes cuidam de apresentacao e interacao.
- Permissao visual pode esconder botoes, mas a permissao real fica em APIs, services e RLS.
- Evitar acesso direto ao banco a partir de componentes.
- Manter componentes pequenos e alinhados ao estilo do Burgarelli Contabil.
