# Lib

Infraestrutura compartilhada do projeto.

Conteudos esperados:

- Clientes Supabase.
- Validacao de variaveis de ambiente.
- Headers de seguranca.
- Rate limiting.
- Logs estruturados.
- Sanitizacao.
- Helpers de request e resposta.

Nao colocar regra de negocio especifica de dominio aqui. Se a regra pertence a empresas, billing, auditoria, convites ou RBAC, ela deve ficar em `modules/`.
